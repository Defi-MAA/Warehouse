仓库管理系统（WMS）

基于 SpringBoot 4.x + Vue3 + Element Plus 开发的轻量级企业级仓库管理系统，支持入库、出库、库存管理、盘点、权限管理等功能。

项目简介

本仓库管理系统适用于中小企业、工厂、电商仓储等场景，实现仓储流程数字化、库存实时监控、单据自动化管理，提升仓库作业效率。

技术栈

后端

核心框架：SpringBoot 4.x

安全框架：Spring Security 4.x

ORM：MyBatis / MyBatis-Flex

数据库：MySQL 5.7 / 8.0

缓存：Redis

工具：Lombok、Hutool、FastJSON

前端

框架：Vue 3

构建工具：Vite

UI 组件：Element Plus

网络请求：Axios

状态管理：Pinia

路由：Vue Router 4.x

功能模块

系统管理：用户管理、角色管理、菜单权限、日志管理

基础资料：商品管理、分类管理、仓库 / 库位管理、供应商 / 客户管理

入库管理：采购入库、退货入库、入库单据

出库管理：销售出库、领料出库、出库单据

库存管理：实时库存、库存流水、库存预警

库存盘点：盘点单、盘点结果、库存修正

数据中心：数据导入导出、报表统计
