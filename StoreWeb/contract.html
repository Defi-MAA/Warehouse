隐私政策

最后更新时间：2025年01月22日

欢迎您使用我们的产品与服务。我们非常重视您的隐私与个人信息保护。本《隐私政策》旨在向您说明我们如何收集、使用、存储、共享您的个人信息，以及您对这些信息享有的控制权。请您在使用我们的产品/服务前，仔细阅读并完整理解本政策全部内容。

一、我们收集的信息

我们仅会在合法、正当、必要的前提下，收集您在使用服务过程中主动提供或因使用服务而产生的信息，包括但不限于：

1. 注册信息：如账号、昵称、手机号码、邮箱等；

2. 设备信息：设备型号、操作系统、唯一设备标识符、IP地址等；

3. 使用信息：浏览记录、点击记录、使用时长、功能偏好等；

4. 您主动上传或填写的其他信息。

二、信息的使用方式

我们会将收集到的信息用于以下用途：

1. 提供、维护、优化我们的产品与服务；

2. 身份验证、客户服务、安全防范；

3. 功能更新、使用统计、体验改进；

4. 预防、排查安全问题及违规行为；

5. 获得您同意的其他用途。

三、信息的共享与披露

我们不会向第三方出售、出租您的个人信息。仅在以下情况下，我们可能会共享或披露信息：

1. 获得您明确同意后；

2. 法律法规要求、政府机关强制要求；

3. 为保护我们及用户的合法权益、安全与财产；

4. 与授权合作方在必要范围内提供，且合作方受严格保密义务约束。

四、信息的存储与保护

我们采用符合行业标准的安全技术与程序保护您的个人信息，防止信息丢失、滥用、篡改或泄露。

您的信息将存储于中国境内，如需跨境传输，我们会单独获得您的授权。

五、您的权利

您可以随时：

1. 访问、更正、更新您的个人信息；

2. 删除您的账号与相关信息（法律法规要求保留的除外）；

3. 关闭某些信息收集权限（可能影响部分功能）；

4. 提出隐私相关的咨询与投诉。

六、未成年人保护

我们非常重视未成年人的个人信息保护。如您为未成年人，请在监护人同意下使用我们的服务。我们不会主动收集未成年人的个人信息，如发现误收，会立即删除。

七、本政策的更新

我们可能根据业务调整或法律法规变化更新本政策。更新后的政策将在页面公布，如继续使用我们的服务，即表示同意更新后的内容。
